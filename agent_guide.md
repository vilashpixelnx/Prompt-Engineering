🧠 Prompt Engineering Landing Page – Project Guide
1. Project Overview
This document defines the complete structure, design rules, tech stack, animation guidelines, and coding standards for building a Prompt Engineering Guide Website using HTML, CSS, and JavaScript.
The goal is to create a modern, clean, and educational website that teaches users how to write effective AI prompts.
2. Project Goal
Create a modern, responsive, animated Prompt Engineering Guide landing page that helps users:
* Learn what prompt engineering is
* Understand techniques
* See examples
* Practice writing prompts
3. Tech Stack
